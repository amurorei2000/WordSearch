## 1. USERS Table ERD

| :users:  | :type:       | :key:       |
| -------- | ------------ | ----------- |
| id       | Integer      | PRIMARY KEY |
| user_id  | varchar(50)  |             |
| password | varchar(100) |             |

## 2. ANSWERS Table ERD

| :answers: | :type:       | :key:       |
| --------- | ------------ | ----------- |
| id        | Integer      | PRIMARY KEY |
| category  | varchar(50)  |             |
| answer    | varchar(100) |             |
